# E-Commerce-Back-End
## Description
This is the back-end part of an E-commerce application. It uses Express.js API to use Sequelize to interact with a MySQL database.

## Installation
Clone the e-commerce-back-end repository to your local computer. Run ```npm install``` in the command line (Win) or terminal (Mac) to install the dependencies.
## Usage
Open Command line (Win) or Terminal (Mac). Go to the e-commerce-back-end folder, go to seeds folder and run ```node index.js``` file to seed the Database, go to the root folder and run ```node server.js``` to start the server. Finally open Insomnia or other platfrom of your choice to hit the routes.
